# Feedback archive

Historical audit and review inputs that were once stored in the repo-root `feedback/` folder live here.

This directory is:

* provenance only
* archive only
* not canonical product truth

Canonical conclusions must land in `products/chummer/*`, not remain trapped in archived feedback markdown.

The repo root stays thin by keeping `feedback/` limited to intake metadata only (`README.md` and `.applied.log`).
