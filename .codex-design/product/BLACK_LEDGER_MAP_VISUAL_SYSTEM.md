# Black Ledger Map Visual System

## Direction

The map should feel like a command table, not a generic dashboard.

## Palette

- background: near-black tactical blue
- grid: muted cyan
- danger: red
- debt/logistics: amber/orange
- magic: violet
- intel: pale steel / cyan

## Faction identity

- Glass Tower Compact: cyan / blue
- Rust Market Syndicate: orange / red
- Ashline Circle: violet / amber
- Neon Docks Union: green / cyan
- Ghostline Network: steel / cyan
- Barrens Free Wardens: amber / green

## Teaser rule

Homepage uses the same geometry, but lighter density:

- district-only teaser
- no full control chrome
- CTA points to `/ledger/map`
